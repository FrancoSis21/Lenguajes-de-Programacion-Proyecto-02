# AnalizadorFP — Programacion Funcional en Python

## Vista General

Proyecto desarrollado en Python utilizando Tkinter y un enfoque basado en Programacion Funcional para el analisis de estudiantes.

La aplicacion permite:

- Cargar archivos CSV
- Analizar rendimiento academico
- Filtrar estudiantes dinamicamente
- Ordenar informacion
- Exportar resultados
- Visualizar estadisticas
- Aplicar ranking academico
- Cambiar temas visuales

El proyecto fue construido aplicando conceptos funcionales reales como:

- map
- filter
- reduce
- composicion de funciones
- closures
- funciones puras
- funciones de orden superior (HOF)
- groupby
- islice
- estado inmutable
